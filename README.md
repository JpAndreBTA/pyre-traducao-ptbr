# Pyre — Tradução PT-BR

Tradução brasileira feita por fã para **Pyre**, da Supergiant Games.

Este projeto aplica localização em **Português do Brasil** para textos, interface e legendas do jogo.

> Projeto não oficial, gratuito e sem vínculo com a Supergiant Games.  
> Pyre e todos os direitos relacionados pertencem aos seus respectivos proprietários.

## Como instalar

1. Feche o Pyre, caso esteja aberto.
2. Execute `Pyre-Traducao-PTBR-Installer-v1.0.2.exe`.
3. Confirme ou procure a pasta de instalação do jogo.
   - Caminho padrão da Steam:

```text
C:\Program Files (x86)\Steam\steamapps\common\Pyre
```

4. Clique em **Instalar**.
5. Abra o jogo.
6. Vá em **Settings / Configurações**.
7. Em **Languages / Idiomas**, selecione:

```text
Português - BR
```

## O que o instalador faz

- Copia os arquivos de tradução PT-BR para a pasta do jogo.
- Cria backup dos arquivos originais antes de substituir.
- Ativa a opção **Português - BR** no menu de idiomas do Pyre.
- Não inclui o jogo completo.
- Não modifica executáveis além do patch necessário para registrar o idioma `pt-BR`.

## Observação sobre antivírus

O instalador é um executável portátil não assinado digitalmente. Por isso, alguns antivírus ou o Windows SmartScreen podem exibir alertas de reputação.

Para conferência, compare o SHA-256 do arquivo baixado com o hash publicado acima.
