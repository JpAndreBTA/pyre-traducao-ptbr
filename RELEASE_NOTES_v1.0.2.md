# Pyre — Tradução PT-BR v1.0.2

Primeira versão pública recomendada da tradução brasileira de Pyre.

## Download

Baixe o arquivo:

```text
Pyre-Traducao-PTBR-Installer-v1.0.2.exe
```

SHA-256:

```text
8B3DF942C0283F6F67669BB2DFEB50D8702DAA2A73FFEF46CD304F45475C0737
```

## Instalação

1. Feche o Pyre.
2. Execute o instalador.
3. Confirme a pasta do jogo.
4. Clique em **Instalar**.
5. Abra o jogo.
6. Vá em **Settings / Configurações > Languages / Idiomas**.
7. Selecione **Português - BR**.

## Notas

- Tradução feita por fã.
- Instalador portátil: não precisa instalar Python, .NET SDK ou outras dependências manualmente.
- O jogo original não está incluído.
- Caso o Windows SmartScreen avise que o app é desconhecido, isso ocorre porque o executável não possui assinatura digital.
